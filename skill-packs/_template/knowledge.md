# My Skill

## Overview
This skill integrates with My Service to provide X capabilities.
When users ask about Y or Z, this skill provides relevant context.

## When to Use
- When the user asks to search or look up something via My Service
- When the user mentions My Service by name
- When the task would benefit from external data from My Service

## How to Use the Actions
You have access to the `my_skill_search` action.
Call it with a query parameter to search My Service.

Example: If user says "look up the weather in London",
you would use my_skill_search(query="weather London").
