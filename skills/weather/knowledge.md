# Weather Lookup

Get current weather conditions and forecasts for any city worldwide.

## Capabilities
- Current temperature, humidity, wind speed, and conditions
- 3-day weather forecasts with high/low temperatures
- Works for any city or location worldwide
- No API key required (uses wttr.in)

## Usage Examples
- "What's the weather in London?"
- "Get the forecast for Dublin"
- "Is it raining in New York?"
- "Weather in Tokyo"

## Data Source
Uses wttr.in for reliable, free weather data.
