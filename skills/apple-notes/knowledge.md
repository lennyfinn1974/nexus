# Apple Notes

Create, list, and search notes in the macOS Notes app.

## Capabilities
- Create new notes with title and content
- List recent notes from any folder
- Search notes by keyword
- Organize into folders

## Usage Examples
- "Create a note called Shopping List with milk, eggs, bread"
- "List my recent notes"
- "Search notes for meeting"

## Requirements
macOS only. Uses AppleScript to interact with Notes.app.
