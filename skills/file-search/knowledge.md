# File Search

Find files on your system using multiple search methods.

## Capabilities
- **Spotlight Search**: Fast indexed search using macOS mdfind
- **Find Files**: Search by filename pattern with directory scope
- **Content Search**: Search inside files for text patterns

## Usage Examples
- "Find all Python files in my project"
- "Search for files named config"
- "Search file contents for 'API_KEY'"
- "Locate my resume"

## Methods
- `spotlight_search` uses macOS Spotlight for fastest results
- `find_files` uses the find command for pattern matching
- `search_file_contents` uses grep for text within files
