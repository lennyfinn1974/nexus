# Apple Calendar

View and create events in macOS Calendar.

## Capabilities
- List upcoming events for the next N days
- Create new calendar events with date/time
- List all available calendars

## Usage Examples
- "What's on my calendar this week?"
- "Create an event 'Team Meeting' tomorrow at 2pm"
- "Show my calendars"
- "List events for the next 3 days"

## Requirements
macOS only. Uses AppleScript to interact with Calendar.app.
