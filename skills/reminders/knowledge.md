# Apple Reminders

Manage reminders in the macOS Reminders app.

## Capabilities
- Create new reminders with optional due dates
- List reminders from any list
- Mark reminders as complete
- Organize into lists

## Usage Examples
- "Remind me to call the dentist"
- "Create a reminder to submit report by Friday"
- "List my reminders"
- "Complete the reminder 'Buy groceries'"

## Requirements
macOS only. Uses AppleScript to interact with Reminders.app.
