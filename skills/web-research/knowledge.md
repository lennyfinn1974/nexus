# Web Research

Fetch web pages, search the web, and extract information.

## Capabilities
- Fetch and extract text content from any public URL
- Search the web using DuckDuckGo (no API key needed)
- Extract key content from pages for summarization
- Handle HTML parsing and text extraction

## Usage Examples
- "Search the web for Python 3.14 new features"
- "Fetch the content of https://example.com"
- "Summarize this article: https://..."

## Methods
- `fetch_url` retrieves and parses HTML, returning clean text
- `search_web` uses DuckDuckGo HTML for search results
- `summarize_page` fetches and extracts the key points
