# Things 3 Task Manager

Manage tasks in Things 3 for Mac using AppleScript and URL schemes.

## Capabilities
- Create new to-dos with notes, lists, and due dates
- List tasks from Today, Inbox, Upcoming, or Anytime
- Mark tasks as complete

## Usage Examples
- "Add a task: Review PR #42"
- "Create a todo to buy groceries for today"
- "Show my tasks for today"
- "Complete the task 'Send invoice'"

## Requirements
Things 3 must be installed on macOS.
