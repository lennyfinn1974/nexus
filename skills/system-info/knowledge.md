# System Information

Monitor system resources including CPU, memory, disk, processes, and network.

## Capabilities
- CPU usage, core count, and frequency
- Memory usage (total, available, used percentage)
- Disk usage for all mounted partitions
- Top processes sorted by CPU or memory
- Network interface details and addresses
- System uptime and OS information

## Usage Examples
- "Show system info"
- "How much disk space is left?"
- "List top processes by memory"
- "Show network info"

## Requirements
Uses the psutil library for cross-platform system monitoring.
