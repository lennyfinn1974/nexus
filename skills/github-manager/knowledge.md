# GitHub Manager

Manage GitHub repositories and pull requests via the gh CLI.

## Capabilities
- Clone repositories to local directory
- List your repositories sorted by stars or update date
- List and view pull requests
- View PR diffs and review status

## Usage Examples
- "Clone fastapi/fastapi"
- "List my repos"
- "Show PRs for my-org/my-project"
- "View PR #42 in lennyfinn1974/nexus"

## Requirements
Requires `gh` CLI to be installed and authenticated.
