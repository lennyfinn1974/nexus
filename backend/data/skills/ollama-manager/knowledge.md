# Ollama Model Manager

Manage local AI models through the Ollama runtime.

## Capabilities
- List all installed models with sizes
- Pull new models from the Ollama library
- Remove models to free disk space
- Inspect model details (parameters, size, quantization)
- Check which models are currently loaded in memory

## Usage Examples
- "List ollama models"
- "Pull llama3 model"
- "Remove qwen2.5:7b"
- "Show info for kimi-k2.5:cloud"
- "What models are running?"

## Requirements
Requires the Ollama CLI to be installed and the Ollama server running.
