# Currency Converter

Convert between 200+ world currencies using real-time exchange rates.

## Supported Currencies
- USD (US Dollar), EUR (Euro), GBP (British Pound)
- JPY (Japanese Yen), CAD (Canadian Dollar), AUD (Australian Dollar)
- CHF (Swiss Franc), CNY (Chinese Yuan), and many more

## Usage Examples
- Convert 100 USD to EUR
- Get current USD to GBP exchange rate
- Check rates for any major world currency

## Data Source
Uses exchangerate-api.com for reliable, up-to-date exchange rates.
