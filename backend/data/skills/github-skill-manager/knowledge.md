# Apple Notes Integration

This skill provides integration with Apple Notes on macOS through AppleScript automation.

## Features

- Create new notes in any folder
- List all notes or notes from specific folders
- Search notes by title or content
- Retrieve note content

## Requirements

- macOS system
- Apple Notes app installed
- AppleScript execution permissions

## Security

Uses AppleScript which requires system permissions. The skill will prompt for access to Notes app on first use.
